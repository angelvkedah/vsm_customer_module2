# VSM — Система анализа диагностических сообщений ВСМ

Веб-приложение для анализа диагностических сообщений высокоскоростного подвижного состава, построения хронологии событий и автоматизированного формирования эксплуатационных протоколов с использованием локальной LLM-модели.

---

## Возможности

- Анализ диагностических сообщений поездов
- Построение временной хронологии событий
- Интеграция локальной LLM-модели
- Экспорт результатов в DOCX / XLSX / CSV
- Декодирование кодов сообщений
- Веб-интерфейс на базе Streamlit

# Установка модели
Проект использует локальную модель Qwen через `llama-cpp-python`.

## Скачать модель

Необходимо скачать модель:

`Qwen2.5-3B-Instruct-GGUF`

Ссылка:
https://huggingface.co/Qwen/Qwen2.5-3B-Instruct-GGUF

Здесь оставлю ссылку на статью, там классно описано почему выбрана именно эта модель
https://habr.com/ru/articles/993312/
---

## Что выбрать при скачивании

В списке файлов выбрать:

- `4-bit`
- `Q4_K_M`

Файл:

```text
qwen2.5-3b-instruct-q4_k_m.gguf
```

---

## Размещение модели

После скачивания поместить файл модели в директорию:

```text
models/qwen/
```

Итоговый путь:

```text
models/qwen/qwen2.5-3b-instruct-q4_k_m.gguf
```
